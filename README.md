# TCA.xctemplate

Installation:

```sh
mkdir -p ~/Library/Developer/Xcode/Templates/Project\ Templates
cd ~/Library/Developer/Xcode/Templates/Project\ Templates
git clone git@github.com:adam-zethraeus/TCA.xctemplate.git
```
